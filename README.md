Once better-agentic-ide is running, you will see a clean workspace.

- **Create a New Project**: Click on “File” in the top menu, then “New Project”. Enter a name and location for your project folder.
- **Open Existing Files**: Use “File” > “Open” to browse and load your code or text files into the editor.
- **Edit Files**: Click on any open file to edit the text. The app highlights code to make it easier to read.
- **Save Your Work**: Click “File” > “Save” or press Ctrl+S to save changes.
- **Run Basic Checks**: Use the built-in tools found under the “Tools” menu to check your code for common errors.
- **Search Function**: Press Ctrl+F to search for specific words in your files.
- **Settings**: Customize your editor appearance and behaviors via “Options” under the “Tools” menu.
